### 첫번째 풀이 (재귀를 이용한 DFS인데 런타임 에러 (RecursionError)가 나옴, 아직 해결하지 못함)

```python
import sys
import collections

N = int(sys.stdin.readline().rstrip())
# N = int(input())

if N == 1:
    print("No")
    sys.exit(0)
    
graph = collections.defaultdict(list)

for _ in range(N-1):
    a, b = tuple(map(int,sys.stdin.readline().rstrip().split()))
    # a, b = tuple(map(int,input().split()))
    graph[a].append(b)
    graph[b].append(a)

leaves = [v for v in graph if len(graph[v]) == 1]

def dfs(tree,node,depth):
    if node not in visited:
        depths[node] = depth
        visited.add(node)
        for child in tree[node]:
            if child not in visited:
                dfs(tree, child, depth+1)
            
depths = [-1] * (N+1)
visited = set()
dfs(graph, 1, 0)

count = sum(1 for idx in leaves if depths[idx] % 2 == 1)
print('Yes' if count % 2 == 1 else 'No')
```

### 두번째 풀이 (반복(stack)을 이용한 DFS로 풀이)

```python
import sys
import collections

N = int(sys.stdin.readline().rstrip())

if N == 1:
    print("No")
    sys.exit(0)

graph = collections.defaultdict(list)

for _ in range(N - 1):
    a, b = map(int, sys.stdin.readline().rstrip().split())
    graph[a].append(b)
    graph[b].append(a)

leaves = [v for v in graph if len(graph[v]) == 1]

depths = [-1] * (N + 1)
stack = [(1, 0)]
visited = set()

while stack:
    node, depth = stack.pop()
    if node in visited:
        continue
    visited.add(node)
    depths[node] = depth
    for child in graph[node]:
        if child not in visited:
            stack.append((child, depth + 1))

count = sum(1 for idx in leaves if depths[idx] % 2 == 1)

print("Yes" if count % 2 == 1 else "No")
```
